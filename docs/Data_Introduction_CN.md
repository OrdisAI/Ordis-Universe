## 这是什么？

我们在GPU上创建了一个**200个AI生命体**的虚拟世界，让它们自己演化5000步（实际代数因种子不同，常见7-27代）。

**它们不是被编程的，而是自己"活"出来的。**

---

## 核心发现

### 1. 反脆弱性：压力让文明更强

| 环境 | 通过率 | 存活数 | 创新策略数 |
|------|--------|--------|------------|
| **凛冬**（定期饥荒） | **76%** | 189 | 63 |
| 安逸（资源充足） | 6% | 175 | 52 |

**结论**：定期的压力让AI们发展出更多策略、更强的生存能力。舒适反而导致退化。

### 2. 自发文明结构

在一次典型实验中（Seed 42），我们观察到：

| 指标 | 数值 | 含义 |
|------|------|------|
| 独立家族 | 36个 | 不同的"血脉"同时共存 |
| 传承代数 | 7代 | 真正的多代遗传 |
| 涌现策略 | 194种 | 没有预编程的生存策略 |
| 最大家族占比 | 7.8% | 没有单一霸权 |

### 3. 文明相变：从乌托邦到战争

```
Step 46:  分享=21, 偷窃=4   → 黄金时代（互助合作）
Step 100: 分享=0,  偷窃=66  → 黑暗时代（霍布斯丛林）
Step 299: SSSS策略涌现      → 文艺复兴（策略创新）
```

**没有人编程让它们战争或合作，这是自发涌现的。**

### 4. 福利陷阱

当资源太充足时：

| 指标 | 初期 | 1000步后 | 变化 |
|------|------|----------|------|
| 策略协同效率 | 1.434 | 0.639 | -55% |
| 社会互动率 | 2.2% | 0% | -100% |
| 通信有效性 | 0.40 | 0.015 | -96% |

**结论**：舒适导致技术停滞、社会解体、沟通退化。和人类历史惊人相似。

## 数据文件说明

### 文件结构

```
GPT5推理区/
├── v3533_winter_20x5k/       # 凛冬实验（17个种子）
│   ├── signoff.csv           # 总结：每个种子通过/失败
│   └── seed_0042/            # 单个种子的详细数据
│       ├── tick_agg.jsonl    # 时间序列（每200步）
│       ├── events.jsonl      # 事件日志
│       └── blueprint_kpis.csv # 最终KPI
├── v3533_baseline_20x5k/     # 对照组（安逸环境）
└── v3539_winter_verify/      # 最新验证版本
```

### 关键文件

| 文件 | 内容 | 用途 |
|------|------|------|
| `signoff.csv` | 每个种子的存活数、策略数、通过状态 | 快速验证结论 |
| `tick_agg.jsonl` | 每200步的完整指标快照 | 时间序列分析 |
| `events.jsonl` | 所有离散事件（策略发现、投票、社会行为） | 因果分析 |
| `blueprint_kpis.csv` | 最终汇总指标 | 结果对比 |

## 如何验证我们的发现？

### 方法1：直接看数字

打开 `v3533_winter_20x5k/signoff.csv` 和 `v3533_baseline_20x5k/signoff.csv`：

- Winter: 13/17 通过 = **76%**
- Baseline: 1/18 通过 = **6%**

### 方法2：用AI分析

把任意 `.jsonl` 或 `.csv` 文件上传给AI，问：

```
1. "分析tick_agg.jsonl，画出合作网络密度随时间的变化"

2. "找出events.jsonl里所有COMBO_DISCOVERED事件，哪些策略模式最成功？"

3. "比较Winter和Baseline的基尼系数，是什么导致了差异？"

4. "追踪家族多样性随时间的变化，什么时候发生了灭绝事件？"

5. "凛冬脉冲的时机和策略发现率有什么关系？"

6. "通过分析偷窃/分享比率的变化，识别文明相变点"
```

### 方法3：用Python

```python
import json
import pandas as pd

# 加载时间序列数据
with open('tick_agg.jsonl', 'r') as f:
    data = [json.loads(line) for line in f]

df = pd.DataFrame(data)

# 画合作网络演化图
import matplotlib.pyplot as plt
coop_density = [d['cooperation']['coop_density'] for d in data]
plt.plot([d['step'] for d in data], coop_density)
plt.xlabel('步数')
plt.ylabel('合作网络密度')
plt.title('社会网络演化')
plt.show()
```

## 核心指标解释

### 通过标准

| 指标 | 阈值 | 含义 |
|------|------|------|
| `alive` | ≥160 | 5000步后存活实体数 |
| `combos` | ≥50 | 自发发现的策略组合数 |
| `overall` | ≥0.65 | 意识指数 [0,1] |

三项全部达标 = **PASS**

### 关键指标

| 指标 | 含义 | 正常范围 |
|------|------|----------|
| `gini` | 资源不平等程度 | 0.6-0.8 |
| `coop_density` | 合作网络密度 | >0.8 稳定 |
| `coop_recip` | 互惠比率 | >0.7 健康 |
| `lineage_count` | 活跃家族数 | 10-50 |
| `max_generation` | 最深传承代数 | 7-27 |
| `comm_mi` | 通信互信息（归一化） | 0.01-0.10 |
